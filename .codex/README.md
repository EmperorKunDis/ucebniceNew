# UcebniceNew Codex Configuration

This directory contains project-scoped Codex configuration.

- `config.toml` sets safe defaults for this repository.
- `rules/default.rules` controls command approval behavior for VPS deploy, database operations, and destructive Git operations.

Do not store personal credentials, API keys, provider settings, auth tokens, telemetry keys, `.env` data, or machine-local secrets here.

Project instructions for agents belong in the root `AGENTS.md`; architecture navigation belongs in `_AI_INDEX.md` and the `_AI.md` files throughout the repo.
