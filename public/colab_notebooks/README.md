# Colab Notebooks

Tato složka obsahuje Jupyter notebooky pro Google Colab, které doplňují jednotlivé kapitoly kurzu.

## Použití

Notebooky lze:

1. **Spustit přímo v Google Colab** - Klikněte na odkaz "Spustit v Colab" na stránce kapitoly
2. **Stáhnout lokálně** - Použijte odkaz "Stáhnout sešit" a spusťte v Jupyter Notebook

## Struktura

Každá kapitola má svůj vlastní notebook s názvem ve formátu:

- `kapitola_01_úvod_do_terminálu_a_příkazové_řádky.ipynb`
- `kapitola_02_instalace_vývojových_nástrojů.ipynb`
- atd.

## Poznámka

Notebooky jsou primárně dostupné přes GitHub repozitář a Google Colab odkazy v aplikaci.
Pokud chcete notebooky používat lokálně, můžete je stáhnout z:

- GitHub repozitář: https://github.com/EmperorKunDis/JupyterNotebooks
- Přímo z Google Colab pomocí odkazů v kapitolách

## Instalace Jupyter Notebook (lokální použití)

```bash
pip install jupyter notebook
jupyter notebook
```

## Závislosti

Notebooky používají různé Python knihovny:

- `numpy` - Numerické výpočty
- `pandas` - Práce s daty
- `matplotlib` - Vizualizace
- `scikit-learn` - Strojové učení
- `tensorflow` / `pytorch` - Deep learning
- A další...

Detaily k závislostem najdete v každém notebooku.
