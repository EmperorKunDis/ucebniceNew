# 3D Assets

This directory contains 3D model files used by the FluidGlass component.

## Required Files:
- `bar.glb` - 3D bar model
- `lens.glb` - 3D lens model  
- `cube.glb` - 3D cube model

These GLB files should be exported 3D models that can be loaded by Three.js.

## Temporary Solution:
For now, you can use any simple GLB files or create basic shapes using Blender and export them as GLB.
