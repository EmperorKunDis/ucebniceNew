# Pokročilé kapitoly (41-50): Reálné využití LLM

Těchto 10 kapitol navazuje na základních 40 kapitol a zaměřuje se na praktické využití velkých jazykových modelů (LLM) v reálných projektech.

## Přehled kapitol

### 41. Prompt Engineering Fundamentals

**Obsah:** Jak efektivně komunikovat s LLM, struktura promptů, few-shot learning, chain-of-thought, systémové prompty.
**Projekt:** Vytvořit sadu optimalizovaných promptů pro různé úlohy.
**NotebookLM:** Podcast o historii a vývoji prompt engineeringu.
**Colab:** Experimentování s různými prompt technikami.

### 42. Embeddings a Vektorové Databáze

**Obsah:** Co jsou embeddings, jak fungují, vektorové databáze (Pinecone, Chroma, Weaviate), similarity search.
**Projekt:** Vytvořit semantic search engine pro vlastní dokumenty.
**NotebookLM:** Jak embeddings změnily AI.
**Colab:** Práce s OpenAI embeddings a ChromaDB.

### 43. RAG (Retrieval-Augmented Generation)

**Obsah:** Architektura RAG, chunking strategie, retrieval techniky, reranking, hybridní search.
**Projekt:** Q&A systém nad vlastní knowledge base.
**NotebookLM:** RAG vs fine-tuning - kdy co použít.
**Colab:** Implementace kompletního RAG pipeline.

### 44. Fine-tuning Modelů

**Obsah:** Kdy a proč fine-tunovat, příprava dat, LoRA/QLoRA, evaluace, deployment.
**Projekt:** Fine-tuning modelu na specifickou doménu.
**NotebookLM:** Příběhy úspěšných fine-tuned modelů.
**Colab:** Fine-tuning s Hugging Face.

### 45. Agenti a Tool-Use

**Obsah:** LLM jako mozek agenta, function calling, ReAct pattern, plánování, multi-agent systémy.
**Projekt:** AI asistent který používá externí nástroje (web, kalkulačka, API).
**NotebookLM:** Budoucnost AI agentů.
**Colab:** Implementace agenta s LangChain.

### 46. Multi-modal LLM (Vision, Audio)

**Obsah:** GPT-4V, LLaVA, Whisper, image understanding, audio transcription, video analysis.
**Projekt:** Aplikace která rozumí obrázkům i textu.
**NotebookLM:** Jak multi-modal AI mění průmysl.
**Colab:** Práce s vision modely.

### 47. LLM v Produkci

**Obsah:** API design, caching, rate limiting, fallbacks, monitoring, cost optimization, latency.
**Projekt:** Production-ready API wrapper pro LLM.
**NotebookLM:** Lekce z nasazení LLM v enterprise.
**Colab:** Best practices pro produkční deployment.

### 48. Evaluace a Benchmarking

**Obsah:** Metriky (BLEU, ROUGE, perplexity), human eval, A/B testing, regression testing.
**Projekt:** Evaluační framework pro vlastní use case.
**NotebookLM:** Jak velké firmy testují AI.
**Colab:** Implementace eval pipeline.

### 49. Safety a Alignment

**Obsah:** Prompt injection, jailbreaks, guardrails, content filtering, RLHF, constitutional AI.
**Projekt:** Implementace bezpečnostních vrstev pro chatbot.
**NotebookLM:** Etika a bezpečnost AI.
**Colab:** Testování a zabezpečení LLM aplikace.

### 50. Vlastní AI Asistent od A do Z

**Obsah:** Integrace všeho předchozího - kompletní projekt od návrhu po deployment.
**Projekt:** Plně funkční AI asistent s RAG, tools, UI, a production deployment.
**NotebookLM:** Příběh vytvoření AI asistenta.
**Colab:** Step-by-step implementace.

## Požadavky na obsah každé kapitoly

1. **Teorie** (4-6 stran)
   - Detailní vysvětlení konceptů
   - Historický kontext
   - Reálné příklady použití

2. **Praktická část** (6-10 stran)
   - Kód s komentáři
   - Step-by-step tutoriál
   - Troubleshooting

3. **Projekt** (2-4 strany)
   - Jasné zadání
   - Kritéria hodnocení
   - Bonusové úkoly

4. **Kvíz** (10 otázek)
   - Mix faktických a konceptuálních
   - Vysvětlení správných odpovědí

5. **NotebookLM podcast** (~15 minut)
   - Engaging narativ
   - Reálné příklady
   - Expert insights

6. **Colab notebook**
   - Interaktivní kód
   - Postupné úkoly
   - Checkpointy

---

## Časová náročnost

Každá kapitola je navržena na **4-6 hodin** aktivního studia, což je 2-3x více než základní kapitoly.

Celkem: **40-60 hodin** pro všech 10 kapitol.
